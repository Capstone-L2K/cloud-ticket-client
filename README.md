# 📍 서비스 소개

#### " 빠르고 쉽게 사용하는 예약 서비스, 클라우드 티켓 🎫" 

[시연영상](https://youtu.be/OyW6q72fa74)
 
소규모 행사를 주최해보신 적이 있으신가요? 행사 예약 및 접수부터 입장 검사 확인까지 일련의 과정을 사람이 수동으로 처리하여 주최측과 참가측 모두 꽤나 수고스러운 절차를 거쳐야합니다. 그렇다고 기존의 티켓 플랫폼을 이용하기에는 주로 기업에서 주최하는 대규모 영리 행사에 초점이 맞춰져  있어 소규모 행사 주최자가 이용하기에 부적절합니다. 클라우드 티켓에서는 소규모 행사를 대상으로 행사 주최 및 홍보, 티켓 구매 및 검표부터 행사 인원 관리까지 행사 주최자가 단 하나의 플랫폼에서 쉽고 편리하게 행사를  관리할 수 있게 합니다.
